# 隐私政策

## OpenClash 规则助手

最后更新: 2025-12-06

### 数据收集

本扩展**不收集**任何用户数据。

### 数据存储

所有数据均存储在您的浏览器本地:
- 路由器配置信息(IP、用户名、密码)
- 规则列表
- 用户偏好设置

这些数据仅存储在您的设备上,不会上传到任何服务器。

### 权限说明

- **activeTab**: 获取当前网页域名
- **tabs**: 管理扩展页面标签
- **contextMenus**: 右键菜单快捷操作
- **storage**: 本地存储配置和规则
- **notifications**: 操作结果通知
- **optional_host_permissions**: 连接您的路由器(需要用户授权)

### 网络请求

本扩展仅与以下地址通信:
- 您配置的路由器 IP 地址
- 当前访问的网站(用于检测连接状态)

不会向任何第三方服务器发送数据。

### 联系方式

如有疑问,请通过 GitHub Issues 联系。
