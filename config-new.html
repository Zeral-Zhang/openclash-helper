Cloudflare Worker 部署教程

1. 注册 Cloudflare 账号
访问 dash.cloudflare.com 注册免费账号
2. 创建 KV 命名空间
Workers & Pages → KV → Create a namespace → 命名为 RULES
3. 创建 Worker
Workers & Pages → Create application → Create Worker → 命名后点击 Deploy → Edit code
4. 粘贴 Worker 代码
删除编辑器中所有代码，粘贴以下内容，点击 Save and Deploy
5. 绑定 KV
点击上方 Bindings 标签页 → Add binding → 从左侧列表选择 KV Namespace
Variable name: RULES，选择刚创建的命名空间，点击 Deploy
6. 设置 API Secret
点击上方 Settings 标签页 → Variables and Secrets → Add
Variable name: API_SECRET，Value: 设置一个密码，勾选 Encrypt，点击 Deploy
7. 获取 Worker URL
返回 Worker 详情页，复制 Preview 下方的 URL（如 https://xxx.workers.dev）
